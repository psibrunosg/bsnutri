# 11 - IA para rascunhos revisaveis

**What to build:** adicionar assistente de IA apenas para gerar rascunhos revisaveis de resumo, orientacao e sugestao de plano a partir de dados ja estruturados.

**Blocked by:** 01 - Base versionada de formularios de anamnese; 10 - Biblioteca profissional de conteudos.

**Status:** ready-for-agent

- [ ] IA gera resumo da anamnese como rascunho.
- [ ] IA sugere orientacao curta a partir de alerta ou meta.
- [ ] IA sugere estrutura inicial de plano sem publicar.
- [ ] Profissional precisa revisar e confirmar qualquer uso.
- [ ] Prompt nao inclui dado fora do paciente e organizacao autorizados.
- [ ] Ha log de uso sem registrar segredo ou dado sensivel desnecessario.
