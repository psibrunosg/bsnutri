# 06 - Editor com periodizacao e modos de prescricao

**What to build:** evoluir o editor para diferenciar dias de treino, descanso, fim de semana ou rotina especial, com refeicoes calculadas, qualitativas ou hibridas.

**Blocked by:** 04 - Biblioteca de modelos de plano.

**Status:** ready-for-agent

- [ ] Plano permite tipos de dia com rotulos claros.
- [ ] Refeicao pode ser calculada ou qualitativa.
- [ ] Paciente ve apenas o dia correto no portal.
- [ ] Profissional duplica dia ou refeicao com um clique.
- [ ] Publicacao preserva periodizacao no snapshot.
- [ ] Ha teste de portal para dia vigente.
