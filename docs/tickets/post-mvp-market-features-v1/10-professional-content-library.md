# 10 - Biblioteca profissional de conteudos

**What to build:** criar biblioteca da clinica para orientacoes, materiais educativos, receitas e protocolos internos reutilizaveis.

**Blocked by:** None - can start after roles are stable.

**Status:** ready-for-agent

- [ ] Profissional cria item de biblioteca com tipo e tags.
- [ ] Item pode ser usado como orientacao em plano ou follow-up.
- [ ] Item publicado tem versao imutavel.
- [ ] Item arquivado nao some do historico ja entregue.
- [ ] Paciente ve apenas conteudos entregues a ele.
- [ ] Ha teste SQL de isolamento por organizacao.
