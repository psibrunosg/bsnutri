# Renders do catálogo

Os arquivos desta pasta são WebP públicos, servidos pelo Vite e versionados no GitHub.

`peixe-grelhado-legumes.webp` vem de [Foodiesfeed](https://www.foodiesfeed.com/pt/free-food-photo/grilled-fish-with-potatoes-and-frozen-vegetables), foto de Antonio Cavalcanti sob licença CC0.

Para um novo item, salve `slug-do-item.webp` nesta pasta e informe `/food-renders/slug-do-item.webp` no catálogo.
